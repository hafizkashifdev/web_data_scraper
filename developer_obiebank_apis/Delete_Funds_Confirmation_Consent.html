Funds Confirmation Consent Request

POST /funds-confirmation-consents

The API allows the CBPII to ask an ASPSP to create a new funds-confirmation-consent resource

Request Headers

AttributeTypeConditionDescription
AuthorizationStringMandatoryAn Authorisation Token as per https://tools.ietf.org/html/rfc6750
x-fapi-customer-ip-addressStringOptionalThe PSU's IP address if the PSU is currently logged in with the TPP.
x-fapi-interaction-idStringOptionalAn RFC4122 UID used as a correlation id.
x-customer-user-agentStringOptionalIndicates the user-agent that the PSU is using.

Request Body

AttributeTypeConditionDescription
DataOBFundsConfirmationConsent1ParamMandatory

Response Headers

AttributeTypeConditionDescription
x-fapi-interaction-idStringOptionalAn RFC4122 UID used as a correlation id.

Response Body

AttributeTypeConditionDescription
