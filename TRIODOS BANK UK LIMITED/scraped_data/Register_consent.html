Register consent

post https://xs2a-sandbox.triodos.com/xs2a-bg//v1/consents

Creates an account information consent resource at the ASPSP regarding access to accounts specified in this request.

boolean
required

"true", if the consent is for recurring access to the account data. "false", if the consent is for one access to the account data.

date
required

This parameter is requesting a valid until date for the requested consent. The content is the local ASPSP date in ISODate Format, e.g. 2017-10-30. If a maximal available date is requested, a date in far future is to be used: "9999-12-31". The consent object to be retrieved by the GET Consent.

int32
required

This field indicates the requested maximum frequency for an access per day. For a one-off access, this attribute is set to "1".

boolean
required

Sessions are not supported. Should be to "false".
